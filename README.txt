
# OpenBCI Feedback Explorer (Starter)

## How to run
1. Open a terminal in this folder.
2. Install dependencies (once):  
   `pip install streamlit pandas plotly`
3. Launch:  
   `python -m streamlit run app.py`

## Notes
- Place your exported CSV files from OpenBCI GUI into a folder like `OpenBCI_Exports/`.
- Use the **Start** page to upload a CSV or load the newest file in that folder.
- Compute the session summary, then explore **Personalization**, **Choose Style**, **Display**, and **Evaluation**.
